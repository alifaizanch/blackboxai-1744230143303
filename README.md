# Pintor Trinitat Vella

## Project Overview
Pintor Trinitat Vella is a personal portfolio website designed for a professional painter operating in Trinitat Vella, Barcelona. The website showcases the painter's work, services, and client testimonials, providing an attractive platform for potential customers to learn about and contact the artist.

## Installation
To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/pintor-trinitat-vella.git
   ```
2. **Navigate into the directory**:
   ```bash
   cd pintor-trinitat-vella
   ```
3. **Open the `index.html` file in your web browser** to view the website:
   ```bash
   open index.html # or use a web server
   ```
   Alternatively, you can serve the project using a local development server like `Live Server` or any other HTTP server.

## Usage
- The homepage welcomes visitors with an introduction to the painter's work.
- Users can navigate to various sections such as "Sobre Mi" (About Me), "Portafoli" (Portfolio), "Serveis" (Services), "Testimonis" (Testimonials), and "Contacte" (Contact).
- Each section provides valuable information and allows potential clients to reach out for services.

## Features
- **Responsive Design**: The website is mobile-friendly, adapting its layout to different screen sizes.
- **Client Testimonials**: Real quotes from satisfied clients to build trust with potential customers.
- **Portfolio Display**: Attractive presentation of projects that highlight the painter’s skills and artistic style.
- **Contact Form**: A simple contact form for inquiries, allowing potential clients to get in touch directly.

## Dependencies
This project utilizes the following external libraries:
- [Tailwind CSS](https://tailwindcss.com/) for styling.
- [Font Awesome](https://fontawesome.com/) for icons.

You don't need to install any packages via npm for this simple static site, but be aware of these links in the `index.html`.
